# Игра "Зеленый свет, красный свет"

## Описание
Это программа для подсчета выбывающих игроков по следующим правилам:
- Если зелёный свет то все проходят (0 выбывают)
- Если красный свет то выбывает каждый игрок, чья скорость не равна 0

## Как запустить программу
1. Убедитесь, что у вас установлена Java.
2. Откройте терминал в папке с программой.
3. Введите команду для компиляции: 'javac Main.java'
4. Затем введите команду для запуска: 'java Main'
